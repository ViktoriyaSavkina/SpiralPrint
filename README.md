The program prints out an two-dimensional array or characters in spiral order.
It takes any NxM array. 

For example 

	var message = new char[,]
        	{
        		{ 'H', 'A', 'V' }, 
                { 'D', 'A', 'E' }, 
                { 'E', 'Y', 'A' }, 
                { 'C', 'I', 'N' }
            };

will be printed as HAVEANICEDAY
